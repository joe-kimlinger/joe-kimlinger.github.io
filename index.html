Link: Student Tracker Node

Book Rental Store

Technologies: Django, Flask, Python, pytest, gitHub workflows

Link: Book Rental Store

The Book Rental Store is an application created with Django that allows users to rent books that are priced based on their genre. A seperate API, created using Flask, provides the same functionality without the web interface. The application was developed over the course of 10 days in late June 2021 and includes GitHub workflows to run tests and linting on pull requests.
